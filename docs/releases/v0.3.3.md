# Valet 0.3.3 - Creation de groupe

## Theme

Creation et controle de groupes de valets.

## Bugs corriges / fonctionnalite

- Ajout du `Pupitre de groupe`, base sur le bloc de lutrin.
- Interface dediee : creation de groupes, liste des valets proches, ajout/retrait par clic.
- Ajout de la `Carte de groupe` liee depuis le pupitre.
- Liaison possible d'une corne de chevre vanilla a un groupe.
- Ordres disponibles : suivre, garde proche, garde large, attaque de zone, rappel.
- Clic droit avec carte/corne liee : suivre.
- Accroupi + clic droit : cycle suivre -> garde proche -> garde large -> rappel.
- Clic sur un monstre : attaque cible.
- Clic sur un bloc : attaque de zone autour du bloc.
- Les combattants et magiciens priorisent les cibles de groupe.
- Les valets non combattants peuvent suivre et etre rappeles sans devenir des combattants.
- Correction apres test log : pas de relance en boucle quand la meme commande est envoyee plusieurs fois.
- Correction apres test log : les non-combattants ne partent plus dans les ordres attaque/garde.
- Correction apres test log : texture zombie-valet manquante ajoutee.
- Correction : les artisans et fermiers fuient les monstres proches.
- Correction : l'item tenu ne bascule plus rapidement entre epee et pioche en groupe.

## Notes corne

En vanilla, les cornes de chevre viennent surtout des chevres et peuvent se trouver dans les coffres d'avant-poste de pillards. Les pillards ne les dropent pas comme loot direct fiable.
