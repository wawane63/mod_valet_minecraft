# Valet 0.3.0 - Decoupage metiers

## Theme

Separation des competences en metiers distincts : artisan, combattant et fermier.

## Bugs corriges / fonctionnalite

- Ajout du `Poste d'artisan`, du `Poste de combattant` et du `Poste de fermier`.
- L'artisan gere minerais, bois, construction et craft.
- Le combattant ne recoit plus d'ordre de recolte et garde uniquement les arbres epee/arc.
- Le fermier garde uniquement l'ordre `Recolter` et les options de champs.
- Ajout d'un arbre fermier dedie : mains vertes, portee, replantation, labour, stockage, intendant.
- Extension combat : portee epee, garde renforcee, tir lointain et volee.
- L'UI affiche seulement les ordres, pages et perks compatibles avec le role actif.
- Les ordres incompatibles sont refuses cote serveur et nettoyes dans le goal si le poste change.
- Les bonus de perks sont limites au metier actif pour eviter les effets croises.
- Ajout des assets, recettes, loot tables, tags et traductions pour les nouveaux postes.
- Correction des postes qui se melangent : le poste stocke du valet est prioritaire sur `JOB_SITE`.
- Les claims `JOB_SITE` de villageois non-valets sur un poste Valet reserve sont nettoyes.
- Le scan d'attribution ignore les doublons quand plusieurs joueurs couvrent le meme villageois.
- Le glow pose par le mod est retire meme si Minecraft a deja retire le metier avant le nettoyage Valet.

## Limites connues

- Test visible en jeu encore a faire sur les trois postes avec plusieurs valets.
